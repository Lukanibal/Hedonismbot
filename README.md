Hedonismbot is [Beelzebot's](https://github.com/Lukanibal/Beelzebot) half-brother.  
  
It is made in Haskell as my way of learning the language.  
Currently it's heavily using [calaimty-tutorial](https://github.com/MorrowM/calamity-tutorial) by MorrowM, thanks MorrowM!  
It's using [Calamity](https://github.com/simmsb/calamity) for the Discord Wrapper, and will use [Ollama](ollama.com) in the future to power it's personality.
